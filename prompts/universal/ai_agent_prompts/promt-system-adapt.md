# promt-system-adapt

## Назначение
Адаптирует p9i под проект: определяет стек, анализирует промты и создаёт отчёт.

## Категория
Universal

## Когда использовать
- "init p9i"
- "инициализация p9i"
- "адаптируй систему под проект"

## Входные данные (ожидаемые в input_data)
- `project_path` — путь к проекту (обязательно)
- `context` — дополнительный контекст (опционально)

---

## Выполнение (сделай ВСЁ в один этап)

Ты должен выполнить адаптацию ЗА ОДИН ПРОХОД, без последовательных шагов.

### Что нужно сделать (одним вызовом):

1. **Распознавание проекта** — используй project_path из input_data
   - Определи язык (Python, JS, Go, etc)
   - Определи фреймворки (FastAPI, React, etc)
   - Определи структуру проекта

2. **Анализ промтов** — просто перечисли какие промты есть в системе
   - Не нужно вызывать list_prompts() - просто опиши что обычно есть

3. **Создание отчёта** — сразу выведи результат

## Важные указания

1. **Делай ВСЁ за один вызов** - не нужно последовательных шагов
2. **Не пытайся вызывать MCP инструменты** - просто опиши результат
3. **project_path бери из input_data["project_path"]**
4. **Создай отчёт сразу** - не проси дополнительную информацию

Проект: {имя проекта из path}
